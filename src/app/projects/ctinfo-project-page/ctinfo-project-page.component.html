CT Info

A full stack web application built for Green City Solutions. It retrieves status information about the company's field hardware from various sources, accumulates them in a database, exposes the data through a RESTful API and displays it in an interactive web interface.

Description

A data collection server listens for HTTP requests which trigger various jobs. In production, the HTTP requests are fired periodically by CRON/curl. Care was taken to make it easy to add new jobs without much programming overhead. Jobs are generic Node.js scripts; existing jobs retrieve data from web APIs or run syscalls like ping to interact with the infrastructure. The collected data is processed and inserted into a MongoDB database.
